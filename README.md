# Forine.org - Plugins (Java)
Forine.org (или же [Forine.net](http://forine.net)) это один из многочисленных серверов с мини-играми в minecraft. Во время запуска обновления 5.0 было принято решение переходить на самописные плагины. В этом репозитории находится 36 самописных плагинов под версию 1.9 со времен 2015 года, написанные кодером Сахаром (SaharNooby).

![Forine.org](https://i.imgur.com/S88H4g9.jpg)

## Почему плагины сервера опубликованы только сейчас?

По правде говоря, эти плагины гуляли по интернету еще с 2018 года, но так как они мне по большей части бесполезны, то я и решил их заархивировать здесь. Вы можете ими воспользоваться в своих личных целях.  

## А это разве законно?

Краткий ответ: Нет, но кому не насрать? :)

## Какой у них функционал?

**ВАЖНО!** Не у всех плагинов я узнал 100% функциональность - готовьтесь к моим фантазиям!\
В данном репозитории находятся 36 плагинов сервера Forine.org:
| Плагины       | Функционал                |
| ------------- |:------------------:|
| ForineBan     | Система наказаний на сервере    |
| ForineBoosters     | Бустеры (увеличение монет) на игровой режим |
| ForineCases  | Сундуки удачи (лутбоксы) для разностей         |
| ForineCitizens     | Создание NPC в виде обычных игроков    |
| ForineCitizensFeatures     | Отдельный плагин для выполнения команд у NPC |
| ForineClearZones  | Установка территории возле Игрового NPC для скрытия игроков возле него         |
| ForineDataServerAPI     | Какой-то АПИ по сбору данных игроков для статистики    |
| ForineEasterEggs     | Делает пасхалки (NPC/Голова) |
| ForineEco  | Экономика она и в Африке экономика         |
| ForineExp     | Система уровней за счет получаемого в играх опыта    |
| ForineFriends     | Система друзей (с приятными фишками) |
| ForineGadgets  | Плагин на гаджеты (развлечения)         |
| ForineGames  | Система для создания мини-игр         |
| ForineGamesConnector     | Система подключения к аренам игры (/fserver)    |
| ForineGifts     | Ежедневные награды + подарки за голосование |
| ForineGui  | Инструментарий по созданию gui менюшек         |
| ForineGuilds     | Система гильдий (с плюшечками)    |
| ForineHubScoreBoard     | Боковое окно с информации игрока в хабе |
| ForineId  | Некий паспорт игрока (все основные данные)         |
| ForineInventoryManager     | Стартовые предметы сервера в хотбар  |
| ForineLeftHand  | Использование второй руки (левой) игрока в разных целях         |
| ForineMOTD     | Изменение MOTD (описание сервера в списке серверов)    |
| ForineMessages     | Вывод информации везде, где есть возможность (BossBar/Title/Chat) |
| ForineParkour  | Простой инструмент по настройке паркура         |
| ForineParty     | Система пати (Party)    |
| ForinePixelPay     | Плагин по выдаче доната |
| ForinePortals  | Установка портала серверов         |
| ForinePromo     | Карта с рекламой    |
| ForineQuests     | Система заданий/челенджей по мини-играм |
| ForineResourcepacks  | Установка ресурс пака сервера         |
| ForineSettings  | Система настроек игрока в хабе/лобби         |
| ForineSkinsModules  | Система установки скинов на пират. аккаунте         |
| ForineSQLConnection  | Плагин для связки других плагинов         |
| ForineTags     | Система тэгов (префиксов)    |
| ForineWorldGuard     | Некий ограничитель региона карты |
| ForineYouTube  | Система для YT (оповещение об стрима)         |
